## Getting Started
```
$ git clone https://github.com/nss-evening-cohort-9/pie/
```
### Prerequisites:

- Install the latest version of [Visual Studio](https://code.visualstudio.com/download)
    
### Installing:

* Set up the _http-server_ on your machine
```
$ npm install -g http-server
```
  * Navigate URL to:
    * `localhost: 5000`
